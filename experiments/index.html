Experiments

Text to Color

This experiment first takes a string and converts it into a hexadecimal color and it's opposite. The experiment is based on the idea that any string of numbers and letters can be viewed as a base 36 number.

Bouncing, Colorful Balls

This experiment draws a canvas that contains any number of (almost) perfectly elastic balls. The experiment then takes the average distance between the balls and puts it through a series of equations and spits out an RGB color to which the background is set, and the balls, the inverse.

Bouncing Text

This experiment draws a canvas that contains any number of strings of text and images. These strings and/or images bounce around the canvas. Type your name into the text box and hit add to see your name. To add an image, start with a backslash (\) and then input the link to the image and hit add.
Fractal Generator

This experiment uses Lindenmayer Systems to draw fractals and space filling curves. There are a lot of examples to show you what it can do. Check it out! Choose some constants, an axiom, and define the rules and see how it turns out. Learn more about L-Systems here or here.
