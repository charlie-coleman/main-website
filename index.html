TrustyClicker

TrustyClicker is a website designed by me, Charlie Coleman. It is based completely off of a popular game: Cookie Clicker combined with my school's IT guy, Mr. Trusty.


The point of the game is to create computers so that you can buy items that will make more computers so you can buy items that make more computers so you can buy more items that make more computers on and on until infinity.

Trusty48

Trusty48 is a game based very closely on the populat game 2048 combined with my school's IT guy, Mr. Trusty.


The goal of 2048 is to combine tiles of equal value by moving all tiles up, down, left, or right. When two tiles of the same value are combined, they will create one tile with a value equivalent to twice the value of the previous tiles. Combine as many tiles as you can before filling the playing field.


Download: .zip or .rar
Just un-zip the files and open index.html in the browser of your choice!
