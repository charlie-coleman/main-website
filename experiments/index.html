Experiments by Charlie Coleman
1. Text to Color!

This experiment first takes a string of any length divides it into three seperate strings. It then takes these three strings and transforms it from a base 36 number into a base 16 number. It then scales the base 16 number down so that is is between 0 and FF (255 in base 10). This makes it possible to concatenate each of those three strings into one hexadecimal color, defining the background. Lots of fun math that makes nice colors (sometimes).

2. Bouncing, Colorful Balls!

This experiment draws a canvas that contains any number of (almost) perfectly elastic balls. You may say, "Well that sounds boring!" But wait, there's more! The experiment then takes the average distance between the balls and puts it through a series of needlessly complex equations (quartic, believe it or not) and spits out an RGB color to which the background is set, and the balls, the inverse. A great follow-up in terms of useless math like the first experiment.

3. Bouncing Text!
This experiment draws a canvas that contains any number of strings of text and images. These strings and/or images bounce around the canvas. Type your name into the text box and hit add to see your name. To add an image, start with a backslash (\) and then input the link to the image and hit add. This will (hopefully) be used at he graduation party for Megan Siembieda and Charlie Coleman.
4. Fractal Generator
This experiment uses Lindenmayer Systems to draw fractals and space filling curves. A lot of examples to show you what it can do. Check it out! Choose some constants, an axiom, and define the rules and see how it turns out. Learn more about L-Systems here or here.
